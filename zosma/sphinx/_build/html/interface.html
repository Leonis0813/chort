アプリ画面¶

  • 表示画面

表示画面¶

機能

  • 画面上部に検索条件を入力するテキストボックスが表示される
  • 検索条件を入力して検索ボタンを押すと家計簿が表示される
  • 家計簿は収入・支出に分けて表示される
  • 何も入力せずに検索ボタンを押すと全家計簿が表示される
  • 家計簿は日付の降順で表示される
  • エラー通知などのメッセージは画面下部に表示される
