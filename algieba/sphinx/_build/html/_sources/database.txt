DBスキーマ定義書
================

**家計簿情報を登録するAccountテーブルを定義する**

- Accountテーブル

+---------------+----------+-----------------------+----------+------------+
| カラム        | 型       | 内容                  | 主キー   | NOT NULL   |
+===============+==========+=======================+==========+============+
| account_type  | STRING   | 収入/支出を表すフラグ |          | ◯          |
+---------------+----------+-----------------------+----------+------------+
| date          | DATE     | 収入/支出があった日   |          | ◯          |
+---------------+----------+-----------------------+----------+------------+
| content       | STRING   | 収入/支出の内容       |          | ◯          |
+---------------+----------+-----------------------+----------+------------+
| category      | STRING   | 収入/支出のカテゴリ   |          | ◯          |
+---------------+----------+-----------------------+----------+------------+
| price         | INTEGER  | 収入/支出の金額       |          | ◯          |
+---------------+----------+-----------------------+----------+------------+
